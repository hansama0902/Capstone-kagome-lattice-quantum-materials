# ========================================
# Installation Instructions:
# ========================================
# 
# Step 1: Install PyTorch first
#   See https://pytorch.org/get-started/locally/ for your specific system
#   
#   Example for CPU:
#   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
#
# Step 2: Install BoTorch and GPyTorch
#   pip install botorch gpytorch
#
# Step 3: Verify installation
#   python3 -c "import torch; print('PyTorch:', torch.__version__)"
#   python3 -c "import botorch; print('BoTorch:', botorch.__version__)"
#   python3 -c "import gpytorch; print('GPyTorch:', gpytorch.__version__)"
#
# ========================================
# Troubleshooting:
# ========================================
#
# Issue: ImportError: cannot import name 'ParameterDict'
# Solution: pip install --upgrade torch
#
# Issue: CUDA version mismatch
# Solution: Check your CUDA version with 'nvcc --version' and install matching PyTorch
#
# Issue: Out of memory on GPU
# Solution: Reduce MC_SAMPLES and RAW_SAMPLES in app_pytorch.py
#
# Issue: Installation fails
# Solution: Use the NumPy version (app.py) instead - it works great!
